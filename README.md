# Dynamic Map

## Development

```bash
# Install dependencies
npm install

# Start Storybook
npm run storybook

# Build the library
npm run build

# Run tests
npm test
```

## Building and Packaging

To create a distributable npm package:

```bash
# Build the library
npm run build

# Create a .tgz tarball
npm pack

# Publish to npm (after bumping version)
npm publish
```
